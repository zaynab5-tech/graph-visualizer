# Graph Algorithm Visualizer

## Description
C++ and SFML project that visualizes graph algorithms like BFS, DFS, Dijkstra, Cycle Detection, and Connected Components step by step.

## Technologies Used
- C++
- SFML
- STL
- Object-Oriented Programming

## Features
- Interactive graph traversal
- Step-by-step execution
- Visual path highlighting
- Weighted graph support
- BFS, DFS, Dijkstra visualization

## How to Run
Compile using:

g++ graphvisualizer.cpp -lsfml-graphics -lsfml-window -lsfml-system
